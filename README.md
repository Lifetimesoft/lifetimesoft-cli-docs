## 🚀 What is lifectl?

**lifectl** is a powerful CLI tool for managing modular applications and agents in a scalable and structured way.

It provides a workflow similar to container-based systems, allowing you to:

* Manage agents and services
* Control versions and environments
* Run and monitor processes
* Extend functionality with modular apps
